fitur
1. sistem database lokal yang terinterasi langsung dengan sqlite
2. terdapat sistem login dan create account
3. dapat melihat waktu secara real-time
4. terdapat greeting sesuai jam yang ada pada halaman dashboard
5. user profile bisa di edit
6. password akun mahasiswa dapat di reset

cara mengkses program
1. buat file flutter baru
2. pastikan sudah memiliki software sqlite
3. konfigurasi pubspec.yaml sesuai dengan versi yang cocok
4. masukan semua kode program ke dalam file .lib
